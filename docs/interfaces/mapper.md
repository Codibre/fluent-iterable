# Interface: Mapper<T, R\>

Represents a mapping operation from type `T` to type `R`.<br>
  Example: ``const userToPrintable: Mapper<User, string> = user => `${user.name} (id: ${user.id})` ``

## Type parameters

Name | Description |
------ | ------ |
`T` | The source type.   |
`R` | The destination type.    |
