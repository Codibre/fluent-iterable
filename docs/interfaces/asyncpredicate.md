# Interface: AsyncPredicate<T\>

Represents an asynchronous predicate on type `T`.<br>
  Example: `const userExists: AsyncPredicate<User> = async user => !!(await getUser(user.id))`

## Type parameters

Name | Description |
------ | ------ |
`T` | The type the predicate is defined on.    |

## Hierarchy

* **AsyncPredicate**

**Returns:** *any*
